# Kaggle Space: Python + Jupyter Notebook

Este repositório é um scaffold para um Kaggle Space com ambiente Python e um Jupyter Notebook pronto para uso.

Arquivos incluídos:

- `requirements.txt` — dependências do ambiente Python usadas pelo Space.
- `space.ipynb` — notebook Jupyter de exemplo.

Como usar no Kaggle Spaces:

1. Crie um novo Space em https://kaggle.com/spaces.
2. Faça upload deste repositório (ou conecte via GitHub) para o Space.
3. Na configuração do Space, selecione "Use a requirements.txt" (Python) — o Kaggle instalará as dependências declaradas.
4. Abra `space.ipynb` no ambiente do Space ou na interface do Jupyter.

Dicas:
- Edite `requirements.txt` para incluir pacotes adicionais que precisar.
- Para notebooks grandes, use datasets do Kaggle vinculados ao Space.
