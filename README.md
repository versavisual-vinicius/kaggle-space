# Kaggle Space: Python + Jupyter Notebook

Este repositório é um scaffold pronto para um Kaggle Space com:

- Ambiente Python configurado via `requirements.txt`
- Notebook Jupyter inicial `space.ipynb`
- Repositório GitHub publicado e pronto para importar no Kaggle

## Conteúdo do repositório

- `requirements.txt` — bibliotecas Python instaladas pelo Kaggle Space.
- `space.ipynb` — notebook de exemplo que valida o ambiente.
- `README.md` — instruções para usar o Space.

## Como usar no Kaggle Spaces

1. Acesse o link direto: `https://www.kaggle.com/spaces/new`
2. Faça login no Kaggle, se necessário.
3. Escolha importar do GitHub e selecione o repositório:
   - `versavisual-vinicius/kaggle-space`
4. Defina o ambiente como `Python`.
5. Certifique-se de que o Kaggle use `requirements.txt` para instalar dependências.
6. Crie o Space.
7. Abra `space.ipynb` e execute as células.

## Sobre o notebook

O `space.ipynb` inclui um exemplo básico de importação de bibliotecas e de criação de um DataFrame com `pandas`.
Ele serve para validar que o ambiente Python está funcionando corretamente no Space.

## Pacotes já disponíveis

- `numpy`

- `matplotlib`
- `scikit-learn`
- `jupyter`

## Personalização

- Adicione pacotes extras no `requirements.txt` conforme necessário.
- Se for usar dados, conecte datasets do Kaggle ao Space para maior performance.
- Para notebooks mais avançados, copie este scaffold e expanda `space.ipynb`.
